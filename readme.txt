Allows you to create your own call to action buttons for your pages.

== Description ==

This plugin allow you to create your own call to action buttons in your style that you can use anywhere on your WordPress site using simple shortcode. 

= Features =

 - Simple for creating Call to Action
 - Enter button details on settings->WP Custom Buttons and generate the shortcode
 - Place the shortcode to your WordPress site

== Installation ==

unzip archive to wp-content/plugins directory, and activate it in Plugins menu in wp-admin
Then go to settings->WP Custom Buttons and enter the button details according to you and generate the shortcode.

== Frequently Asked Questions ==

= how to use this plugin =

Go to Wordpress settings -> WP Custom Buttons and enter the button details according to you and generate the shortcode. Now just copy and paste it anywhere in your WordPress site.
